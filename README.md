# Penguin Day (Flask)

## Descripción

Un Penguin Day at School, donde hagamos una presentación de cosas que aprendimos en el bootcamp, dirigido a chicos de 1ro a 3ro de la media de colegios.

## Tecnologías utilizadas

- Python
  - Flask
- HTML5
- CSS
  - TailwindCSS
- Raspberry Pi
- Sensor DHT11
